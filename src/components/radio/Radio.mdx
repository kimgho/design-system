# Radio

사용자가 여러 옵션 중 하나를 선택할 수 있는 라디오 버튼 컴포넌트입니다.
Compound Component 패턴으로 라벨, 옵션, 설명을 유연하게 구성할 수 있습니다.

## 사용 가이드 (Usage)

Compound Component 패턴으로 각 부분을 자유롭게 조합할 수 있습니다.

```tsx
const [value, setValue] = useState('option1');

<Radio.Root value={value} onValueChange={setValue}>
  <Radio.Label>배송 방법을 선택하세요</Radio.Label>

  <Radio.Item value="standard">
    <Radio.Control />
    <Radio.Indicator />
    <Radio.Text>일반 배송 (무료)</Radio.Text>
  </Radio.Item>

  <Radio.Item value="express">
    <Radio.Control />
    <Radio.Indicator />
    <Radio.Text>빠른 배송 (+3,000원)</Radio.Text>
  </Radio.Item>

  <Radio.Item value="overnight">
    <Radio.Control />
    <Radio.Indicator />
    <Radio.Text>새벽 배송 (+5,000원)</Radio.Text>
  </Radio.Item>

  <Radio.Description>배송비는 결제 시 추가됩니다</Radio.Description>
</Radio.Root>;
```

### 상태 관리

`value`와 `onValueChange`를 사용하여 선택된 값을 관리합니다.

```tsx
const [deliveryMethod, setDeliveryMethod] = useState('standard');

<Radio.Root value={deliveryMethod} onValueChange={setDeliveryMethod}>
  <Radio.Label>배송 방법</Radio.Label>

  <Radio.Item value="standard">
    <Radio.Control />
    <Radio.Indicator />
    <Radio.Text>일반 배송</Radio.Text>
  </Radio.Item>

  <Radio.Item value="express">
    <Radio.Control />
    <Radio.Indicator />
    <Radio.Text>빠른 배송</Radio.Text>
  </Radio.Item>
</Radio.Root>

<p>선택된 배송 방법: {deliveryMethod}</p>
```

### 키보드 네비게이션

`Radio` 컴포넌트는 표준 키보드 네비게이션을 지원합니다:

- **Tab**: 라디오 그룹으로 진입/탈출 (선택된 항목 또는 첫 번째 항목으로 포커스)
- **Arrow Keys** (↑↓←→): 라디오 옵션 간 이동 및 자동 선택
- **Space**: 현재 포커스된 옵션 선택

라디오 그룹은 단일 선택이므로, 그룹 전체가 하나의 Tab stop으로 작동합니다.
