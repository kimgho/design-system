# Input

사용자로부터 텍스트 입력을 받는 폼 컴포넌트입니다.
라벨, 설명, 에러 메시지를 지원하며, 다양한 크기와 상태를 제공합니다.

## Examples

### Status

`Input` 컴포넌트는 기본, 필수(`required`), 에러(`errorText`), 비활성화(`disabled`) 상태를 지원합니다.

{/* 에러 */}

## 사용 가이드 (Usage)

`label`과 `placeholder`를 통해 입력 필드를 구성합니다.

```tsx
<Input label="이메일" placeholder="example@email.com" description="회사 이메일을 입력해주세요" required />
```

### 에러 처리

`errorText`를 전달하면 자동으로 에러 스타일이 적용됩니다.

```tsx
const [error, setError] = useState('');

<Input
  label="비밀번호"
  type="password"
  errorText={error}
  onChange={e => {
    if (e.target.value.length < 8) {
      setError('비밀번호는 8자 이상이어야 합니다');
    } else {
      setError('');
    }
  }}
/>;
```
